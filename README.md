# Curso de Auditoría de Sistemas (Propuesta)

Bienvenidos al espacio complementario de apoyo del curso de **Auditoría de Sistemas**.

Este espacio en GitHub tiene como propósito organizar materiales de apoyo, ejemplos, guías, prácticas, lecturas y recursos complementarios que serán utilizados durante el desarrollo del curso.

## Profesores del curso

- **Rita Lorena Cortés Chavarría, CISA, AIAA, Disciplined Agile Scrum Master**
- **Dr(c). Jonathan Morales Murillo**


## Importante

El **Aula Virtual es el medio oficial del curso**.

En el Aula Virtual se publicarán:

- Avisos oficiales.
- Evaluaciones.
- Entregas.
- Calificaciones.
- Instrucciones formales.
- Comunicaciones institucionales.

GitHub se utilizará únicamente como un **espacio complementario de apoyo** para consultar materiales relacionados con los temas vistos en clase.

## ¿Cómo usar este repositorio?

Para encontrar el material del curso, revise las carpetas organizadas por temas o semanas.

```text (esto se puede cambiar Profesora Rita)
Auditoria-de-Sistemas/
│
├── README.md
├── 01-Conceptos-Generales/
├── 02-Tipos-de-Auditoria/
├── 03-Gobierno-y-Gestion-TI/
├── 04-Fases-de-la-Auditoria/
├── 05-Control-Interno/
├── 06-Riesgos-y-Cumplimiento/
├── Practicas/
├── Lecturas/
├── Recursos/
└── Rubricas/
```


Cada repositorio de GitHub asociado a una clase podrá contener diferentes materiales de apoyo, según el tema que se esté desarrollando.

En estos repositorios podrán encontrar:

- Presentaciones.
- Lecturas.
- Guías de trabajo.
- Prácticas.
- Ejemplos.
- Recursos complementarios.
- Material utilizado durante la clase.

La finalidad es que cada estudiante pueda acceder de forma ordenada únicamente al material correspondiente a la sesión indicada, sin necesidad de descargar nuevamente todos los recursos del curso.

## Recomendación para los estudiantes

Antes de cada clase, se recomienda:

1. Revisar primero el Aula Virtual.
2. Verificar si existen avisos o instrucciones oficiales.
3. Consultar este repositorio de GitHub cuando se indique que hay material de apoyo disponible.
4. Descargar o revisar los archivos correspondientes al tema de la semana.

## Cómo descargar un archivo

Para descargar un archivo específico:

1. Ingrese a la carpeta correspondiente.
2. Haga clic sobre el archivo.
3. Seleccione la opción **Download raw file** o **Raw**.
4. Guarde el archivo en su computadora.

## Cómo descargar todo el material

Para descargar todo el contenido del repositorio:

1. Presione el botón verde **Code**.
2. Seleccione **Download ZIP**.
3. Descomprima el archivo descargado.
4. Abra la carpeta en su computadora.

## Cómo clonar el repositorio usando Git

Si ya tiene Git instalado, puede clonar el repositorio con el siguiente comando:

```bash
git clone URL_DEL_REPOSITORIO
```

Ejemplo:

```bash
git clone https://github.com/usuario/auditoria-de-sistemas.git
```

## Cómo actualizar el material

Si ya clonó el repositorio, puede actualizarlo con:

```bash
git pull
```

Este comando descarga los cambios más recientes agregados al repositorio.

## Nota final

Este repositorio **no sustituye el Aula Virtual**.

Su función es servir como apoyo académico para facilitar el acceso a materiales complementarios del curso.
